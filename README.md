# RateMyBench

RateMyBench is a microservices-based web application where users can share, rate and discuss public benches.

## Project Goal
Build a containerized web application with:
- a frontend
- two backend microservices
- one isolated database per backend
- Docker Compose orchestration
- CI/CD pipeline with security scanning

## Planned Architecture
- Frontend: React + Vite
- API 1: Bench Service
- API 2: Discussion Service
- Database 1: PostgreSQL
- Database 2: PostgreSQL

## Main Features
- List benches
- Add a bench
- View bench details
- Rate benches
- Comment on benches
- Search and sort benches
